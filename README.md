## How to run

1. Clone the repository
```bash
git clone https://github.com/yourusername/brick_breaker.git
cd brick_breake
```

2. Install pygame
```bash
pip install pygame
```

3. Run file
```bash
windows: py breaker.py
mac: python3 breaker.py
```
